## 第 3 章
ファイル操作のコマンド。

### ls
ls の 引数に `*` や `?` を使うことで絞り込みができる。  
`?` は 1 文字のワイルドカード。

#### オプション
- `-a`: 隠しファイルの表示
- `-t`: タイムスタンプでソート
- `-l`: 詳細情報の表示
- `-r`: 逆順にする

### cp, mv
使い方が 2 通りある。  
第二引数（コピー/移動先）が

- ファイル名: そのファイル名（パス）にコピー/移動
- ディレクトリ: そのディレクトリの下にコピー/移動

#### オプション
- `-r`: ディレクトリのコピー/移動
- `-i`: 処理を行うときに確認をする
- `-p`: 元ファイルの情報を保持する (cp)
